
Tinkers' Mastermind(匠魂长阶)
=======

百尺竿头,更进一步.

这是一个匠魂3(Tinkers' Construct 3)的附属模组,致力于写出具有创意的玩法,在匠魂的平台上拓展玩法的深度.

主要活跃分支为1.20.1(neo)forge.

1.0版本的目标:
- 12个匠魂材料以及相关的匠魂工具/护甲
- 3种新工具
- 共计16种新的匠魂属性


该模组参考了工匠巧思(Tinkers' Thinking)的结构来进行框架的搭建.
